# Resources

## Go

- [A Tour of Go](https://go.dev/tour/) — official interactive intro; covers packages, types, functions, concurrency. Start here for any new Go concept.
- [Effective Go](https://go.dev/doc/effective_go) — idiomatic Go conventions from the authors. Reference, not a tutorial — use after the Tour.
- [Go for Python Developers (Ray Thurman)](https://www.raythurman.dev/blog/go-for-python-developers-a-practical-hands-on-guide-to-modern-go) — direct comparisons: static typing, concurrency, error handling, generics.
- [A Gentle Introduction to Go for Python Programmers (KDnuggets)](https://www.kdnuggets.com/a-gentle-introduction-to-go-for-python-programmers) — shorter overview, good for first pass.

## D&D 5e

- [SRD 5.2](https://5e24srd.com/index.html) — the complete free rules reference. Authoritative. Search-friendly.
- [D&D 2024 Free Rules (D&D Beyond)](https://www.dndbeyond.com/posts/1804-start-playing-today-with-the-2024-d-d-free-rules) — 12 base classes with subclasses; slightly more readable than the SRD.
